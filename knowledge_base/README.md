# Инструкция по обновлению Базы Знаний (Knowledge Base)

Данный репозиторий содержит базу знаний для ИИ-ассистента NeoMoney. Основным источником данных является файл `knowledge_base.csv`.

## Как обновлять данные

Для удобства редактирования рекомендуется использовать **Google Таблицы**.

### Пошаговый процесс:
1. **Откройте шаблон** в Google Таблицах (ссылка предоставляется HR-менеджеру).
2. **Внесите изменения**:
   - Добавьте новые строки или отредактируйте существующие.
   - **Важно**: не меняйте названия колонок (`id`, `category`, `keywords` и т.д.).
   - Для ответов используйте краткий стиль, маркированные списки и жирный шрифт для акцентов (подходит для Telegram).
3. **Экспортируйте файл**:
   - `Файл` $\rightarrow$ `Скачать` $\rightarrow$ `Comma-separated values (.csv)`.
4. **Загрузите в репозиторий**:
   - Переименуйте скачанный файл в `knowledge_base.csv`.
   - Замените текущий файл в папке `knowledge_base/` через GitHub.

## Описание полей CSV

| Поле | Описание | Пример |
|---|---|---|
| `id` | Уникальный номер записи | `1`, `2`, `3` |
| `category` | Категория темы | `HR Policies`, `Onboarding` |
| `keywords` | Ключевые слова через запятую для поиска | `отпуск, vacation, даты` |
| `question` | Типовой вопрос сотрудника | `Как оформить отпуск?` |
| `answer` | Краткий структурированный ответ | `1. Зайти в NeoPeople...` |
| `escalation` | Куда направить, если бот не помог | `hr`, `it_support`, `manager` |
| `doc_type` | Тип документа | `procedure`, `guide`, `policy` |
| `department` | Ответственный отдел | `HR`, `IT`, `Security` |
| `date` | Дата последнего обновления | `2026-06-29` |
| `language` | Язык ответа | `ru` |
| `source` | Источник данных | `csv_internal` |

---
*Любые изменения в CSV вступают в силу после перезапуска/обновления индекса бота.*
